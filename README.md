# Dog Image Generator 🐶

A minimalist web application that fetches random dog images from a public API.

## 🚀 Features
*   Simple button to fetch new images.
*   Uses the free [Dog API](https://dog.ceo).
*   No installation required.

## 🛠️ Built With
*   **HTML5** - Page structure.
*   **JavaScript** - Fetching data from the API.

## 📖 How to Use
1.  Download or clone the repository.
2.  Open the `index.html` file in any web browser.
3.  Click the **"Get New Dog"** button to see a random puppy!

## 🔗 API Used
This project connects to:
`https://dog.ceo`
